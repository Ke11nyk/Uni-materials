\doxysubsection*{Linux}

A simple way to build this project is with cmake, clone the repository, cd into the root of the project and run\+:


\begin{DoxyPre}

  {\ttfamily 

    mkdir my\_build

    cmake -\/S . -\/B my\_build

    cd my\_build

    cmake -\/-\/build .

  }

\end{DoxyPre}
